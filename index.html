📖 How to Use oxo-call Skills

oxo-call skills are domain-expert knowledge files that help AI assistants generate accurate bioinformatics commands. Each skill file contains:

  • Concepts: Mental model of how the tool works, input/output formats, and key workflows
  • Pitfalls: Common mistakes, version differences, and how to avoid them
  • Examples: Real-world command examples with detailed explanations

🚀 Loading a Skill in oxo-call

# Download a skill file curl -O https://traitome.github.io/oxo-call-extends/skill/samtools.md # Load into oxo-call oxo-call load-skill samtools.md # Or reference by URL in your AI assistant # The AI will fetch and use the skill automatically
