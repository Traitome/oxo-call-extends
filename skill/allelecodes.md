## Concepts

- **Tool Overview**: allelecodes (v2.1) - Allele Code Assignment Algorithm for cgMLST schemes
- **Core Function**: This tool assigns nearest-neighbor hierarchical codes to allele profiles using cgMLST schemes, as used in public health genomics.
- **Input/Output**: Standard bioinformatics formats
- **Installation**: `conda install -c bioconda allelecodes`
