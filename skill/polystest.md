## Concepts
- **Tool Overview**: PolySTest is a web service (shiny app) and command-line tool for statistical testing, data browsing and interactive visualization of quantitative omics data. It contains multiple statistical tests and a new method to incorporate missing values.
- **Core Function**: Interactive tool for statistical testing, data browsing and interactive visualization of quantitative omics data
- **Input/Output**: Various formats
- **Installation**: `conda install -c bioconda polystest`
