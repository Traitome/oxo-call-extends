## Concepts
- **Tool Overview**: Proper recognition to the [Protein Data Bank](http://mmcif.wwpdb.org/docs/sw-examples/python/html/index.html) where this library for protein structures in the mmCIF format initially came from. It is modified the original library to support python3.
- **Core Function**: A parser module in python for structures of the protein data bank in the mmcif format
- **Input/Output**: Various formats
- **Installation**: `conda install -c bioconda pdbx`

## Pitfalls
