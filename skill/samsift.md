## Concepts

- **Tool Overview**: Advanced filtering and tagging of SAM/BAM alignments using Python expressions. (version 0.3.1)
- **Core Function**: Processes bioinformatics data related to alignment
- **Input/Output**: Standard bioinformatics formats
- **Installation**: `conda install -c bioconda samsift`

## Pitfalls

- **Input Format**: Ensure correct input format.
