## Concepts
- **Tool Overview**: The OMERO.insight Project is a sub-project of the Open Microscopy Environment Project, OME that focuses on delivering a client for the visualization and manipulation of both image data and metadata maintained at an OMERO server site.
- **Core Function**: ImageJ / Fiji plugin to visualize and manipulate both image data and metadata maintained at an OMERO server site.
- **Input/Output**: Depends on tool configuration and data formats.
- **Installation**: `conda install -c bioconda fiji-omero_ij`
