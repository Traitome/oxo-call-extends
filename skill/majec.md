## Concepts

- **Tool Overview**: majec v0.1.4 - MAJEC is a unified framework for quantifying genes, transcript isoforms, and individual transposable element loci from aligned RNA-seq reads (BAM files) in a single analysis. It uses a junction-informed Expectation-Maximization algorithm operating on a joint gene+TE feature space..
- **Core Function**: Momentum-Accelerated Junction-Enhanced Counting for RNA-seq quantification
