## Concepts
- **Tool Overview**: Command-line tools to create, manipulate, and/or analyze phylogenetic trees or pairwise alignments. Does not include "rudisvg" svg viewer.
- **Core Function**: Command-line phylogenetics tools.
- **Input/Output**: BAM/SAM/GFF/GTF
- **Installation**: `conda install -c bioconda phylommand`

## Pitfalls
