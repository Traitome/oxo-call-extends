- **Tool Overview**: oarfish is a tool for fast, accurate and versatile transcript quantification from long-read RNA-seq data
- **Core Function**: Processes bioinformatics data for expression tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda oarfish`
