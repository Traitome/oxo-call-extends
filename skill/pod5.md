## Concepts
- **Tool Overview**: POD5 File Format ---------------- POD5 is a file format for storing nanopore dna data in an easily accessible way. The format is able to be written in a streaming manner which allows a sequencing instrument to directly write the format. Data in POD5 is stored using Apache Arrow, allowing users to consume data in many languages using standard tools. What does this project contain ------------------------------ This project contains a core library for reading and writing POD5 data, and a toolkit for accessing this data in other languages. Documentation ------------- Full documentation is found at https://pod5-file-format.readthedocs.io
