## Concepts
- **Tool Overview**: Euktect uses the HyenaDNA deep learning model to assess the quality of Eukaryotic Metagenome-Assembled Genomes (MAGs). This CPU-only build does not require CUDA or GPU hardware.
- **Core Function**: Eukaryotic MAG quality assessment using HyenaDNA (CPU-only)
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda euktect`
