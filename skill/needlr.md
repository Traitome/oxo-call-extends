## Concepts
- **Tool Overview**: needLR is a command line tool that uses Truvari merging to compare query structural variant (SV) vcfs to our collection of 1000 Genomes Project (1KGP) samples sequenced by Oxford Nanopore Technologies long-read sequencing (ONT LRS).
- **Core Function**: Processes bioinformatics data for variant-calling tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda needlr`

## Pitfalls
- **Version**: Options may vary between versions.
