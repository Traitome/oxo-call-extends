## Concepts
- **Tool Overview**: Earl Grey is a full-automated transposable element (TE) annotation pipeline, leveraging the most widely-used tools and combining these with a consensus elongation process (BEAT) to better define de novo consensus sequences when annotating new genome assemblies.
- **Core Function**: Earl Grey: A fully automated TE curation and annotation pipeline.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda earlgrey`
