## Concepts
- **Tool Overview**: This software and the included database use a standardized algorithm to accurately type both known and unknown Shiga toxin operons from assembled genomic sequence
- **Core Function**: Processes bioinformatics data for assembly tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda ncbi-stxtyper`

## Pitfalls
