## Concepts
- **Tool Overview**: OrientationPy is the pythonic successor to the well-loved OrientationJ Fiji Plugin. It is a library that takes in 2D images or 3D volumes and computes the orientation of the greylevels.
- **Core Function**: Processes bioinformatics data for programming tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda orientationpy`
