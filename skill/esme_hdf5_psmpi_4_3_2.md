## Concepts
- **Tool Overview**: HDF5 (Hierarchical Data Format 5) is a file format and set of tools designed for managing, storing, and organizing large and complex data. The VFD-GDS plugin enables direct GPU-to-storage I/O in HDF5 applications using NVIDIA's GDS API, bypassing CPU buffers for improved performance in GPU workflows.
- **Core Function**: HDF5 - Hierarchical Data Format 5 ((includes VFD-GDS plugin for the MVAPIch version)
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda esme_hdf5_psmpi_4_3_2`
