## Concepts
- **Tool Overview**: The Nextstrain command-line interface (CLI)—a program called nextstrain—aims to provide a consistent way to run and visualize pathogen builds and access Nextstrain components like Augur and Auspice across computing environments such as Docker, Conda, and AWS Batch.
- **Core Function**: Processes bioinformatics data for containerization tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda nextstrain-cli`

## Pitfalls
- **Version**: Options may vary between versions.
