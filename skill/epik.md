## Concepts
- **Tool Overview**: EPIK uses pre-computed phylo-k-mers indexes (see package IPK) to rapidly place large amounts of sequences on a fixed phylogenetic tree. Please cite: doi.org/10.1093/bioinformatics/btad692
- **Core Function**: EPIK is a tool for fast alignement-free phylogenetic placements.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda epik`

## Pitfalls
- **Version**: Options may vary between versions.
