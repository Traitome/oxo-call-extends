- **Tool Overview**: A python package implementing a statistical method for inferring and visualizing gene-flow in spatial population genetic data.
- **Core Function**: Fast Estimation of Effective Migration Surfaces (FEEMS) + admixture (FEEMSmix)
- **Input/Output**: Depends on tool configuration and data formats.
- **Installation**: `conda install -c bioconda feems`
