## Concepts
- **Tool Overview**: NetCDF-Fortran provides Fortran language bindings for the NetCDF library, allowing Fortran programs to read and write scientific data in netCDF format.
- **Core Function**: NetCDF-Fortran - netCDF interface for Fortran
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda esme_netcdf-fortran_openmpi_5_0_7`
