## Concepts
- **Tool Overview**: Java tools for working with NGS data in the BAM format. This package lacks the R dependency that is only required for some metrics tasks. This keeps the size of the package smaller, at the cost of breaking some of Picards's commands. The 'picard' package contains all the necessary dependencies.
- **Core Function**: Java tools for working with NGS data in the BAM format.
- **Input/Output**: BAM/SAM
- **Installation**: `conda install -c bioconda picard-slim`

## Pitfalls
- **Version**: Options may vary between versions.
