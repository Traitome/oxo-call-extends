## Concepts
- **Tool Overview**: FROGS produces an ASVs count matrix from high depth sequencing amplicon data. This is the official release 5.1.0 of FROGS. To fully install FROGS dependencies, please refer to the frogs-conda-requirements.txt and frogsfunc-conda-requirements.txt available at https://github.com/geraldinepascal/FROGS
- **Core Function**: FROGS is a workflow designed to metabarcoding sequence analysis
- **Input/Output**: Depends on tool configuration and data formats.
- **Installation**: `conda install -c bioconda frogs`
