## Concepts
- **Tool Overview**: PLAST is a tool for high-performance local alignment search (DNA/protein), parallellized and optimized to exploit SIMD / multithreading architectures.
- **Core Function**: PLAST: Parallel Local Alignment Search Tool
- **Input/Output**: BAM/SAM
- **Installation**: `conda install -c bioconda plast`
