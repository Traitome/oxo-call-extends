## Concepts
- **Tool Overview**: polyat counts poly-A/T homopolymers across FASTQ reads (>=10/15/20 nt), summarizes per sample, and writes both TSV and HTML reports with interactive filters.
- **Core Function**: Command-line tool to quantify poly-A/T homopolymers within FASTQ sequencing reads.
- **Input/Output**: FASTQ/BAM/SAM
- **Installation**: `conda install -c bioconda polyat`

## Pitfalls
- **Version**: Options may vary between versions.
