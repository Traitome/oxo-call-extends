## Concepts
- **Tool Overview**: fastreeR is a Python+Java CLI tool for calculating distance matrices from VCF or FASTA files and constructing Newick trees using a Java backend and Python frontend.
- **Core Function**: Phylogenetic, distance, and tree-building utilities for VCF and FASTA files.
- **Input/Output**: Depends on tool configuration and data formats.
- **Installation**: `conda install -c bioconda fastreer`
