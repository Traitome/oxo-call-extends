## Concepts
- **Tool Overview**: Java tools for working with NGS data in the BAM format. This package depends on 'r-base' because Picard requires R to run some of its metrics commands. The 'picard-slim' package offers an installation of Picard without the  'r-base' dependency.
- **Core Function**: Java tools for working with NGS data in the BAM format
- **Input/Output**: BAM/SAM
- **Installation**: `conda install -c bioconda picard`

## Pitfalls
- **Version**: Options may vary between versions.
