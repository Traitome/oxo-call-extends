## Concepts

- **Tool Overview**: Raven is an assembler for raw reads generated by the third generation sequencing. (version 1.8.3)
- **Core Function**: Processes bioinformatics data related to assembly
- **Input/Output**: Standard bioinformatics formats
- **Installation**: `conda install -c bioconda raven-assembler`

- **Input Format**: Ensure correct input format.

**Args:** `--help`
**Explanation:** Shows available options and usage information.

### Run assembly
**Args:** `-i reads.fastq -o assembly_dir`
