## Concepts
- **Tool Overview**: `fastppm` (fast perfect phylogeny mixtures) is a C++/Python library for fast estimation of unknown frequency matrices, given variant and total read counts over an n-clonal tree.
- **Core Function**: Fast Perfect Phylogeny Mixture Regression using Tree-Structured Dual Dynamic Programming
- **Input/Output**: Depends on tool configuration and data formats.
- **Installation**: `conda install -c bioconda fastppm`
