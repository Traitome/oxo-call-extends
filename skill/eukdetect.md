## Concepts
- **Tool Overview**: EukDetect detects and quantifies eukaryotic microbes in shotgun metagenomic sequencing data using a curated database of conserved marker genes.
- **Core Function**: Detect eukaryotes from shotgun metagenomic sequencing data
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda eukdetect`
