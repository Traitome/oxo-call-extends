## Concepts
- **Tool Overview**: Pneumo-Typer is a stand-alone perl application, which requires blat, prodigal, NCBI BLAST+, and several perl Modules (GD, GD::SVG, Net::OAuth, JSON, LWP::UserAgent, HTTP::Request::Common, Data::Random, Config::Tiny) to be installed before use.
- **Core Function**: Pneumo-Typer is a high-throughput capsule genotype visualization tool with integrated serotype and sequence type prediction for Streptococcus pneumoniae
- **Input/Output**: FASTA
- **Installation**: `conda install -c bioconda pneumo-typer`

## Pitfalls
