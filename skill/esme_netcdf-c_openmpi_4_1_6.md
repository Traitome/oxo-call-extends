## Concepts
- **Tool Overview**: NetCDF-C is the C implementation of the Unidata's network Common Data Form (netCDF), providing an interface for storing, accessing, and sharing array-oriented scientific data in a platform-independent format.
- **Core Function**: NetCDF-C - netCDF interface for C
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda esme_netcdf-c_openmpi_4_1_6`
