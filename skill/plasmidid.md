## Concepts
- **Tool Overview**: PlasmidID is a mapping-based, assembly-assisted plasmid identification tool that analyzes and gives graphic solution for plasmid identification.
- **Core Function**: Pipeline for plasmid identification and reconstruction
- **Input/Output**: BAM/SAM
- **Installation**: `conda install -c bioconda plasmidid`
