- **Tool Overview**: A package to annotate a pretext snapshot with size and chromosome information (version 1.1.3)
- **Core Function**: Processes bioinformatics data related to annotation
- **Input/Output**: Standard bioinformatics formats
- **Installation**: `conda install -c bioconda pretextannotate`
