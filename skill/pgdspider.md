## Concepts
- **Tool Overview**: PGDSpider is a powerful automated data conversion tool for population genetic and genomics programs. It facilitates the data exchange possibilities between programs for a vast range of data types (e.g. DNA, RNA, NGS, microsatellite, SNP, RFLP, AFLP, multi-allelic data, allele frequency or genetic distances).
- **Core Function**: An automated data conversion tool for connecting population genetics and genomics programs
- **Input/Output**: VCF/GFF/GTF
- **Installation**: `conda install -c bioconda pgdspider`

## Pitfalls
- **Version**: Options may vary between versions.
