## Concepts
- **Tool Overview**: A package for parsing ontologies in the OWL and OBO format into NetworkX graphs
- **Core Function**: Processes bioinformatics data for formatting tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda onto2nx`
