## Concepts
- **Tool Overview**: eveHap is a universal mtDNA (mitochondrial DNA) haplogroup classifier. It classifies ancient and modern DNA samples against phylogenetic haplogroup references, supporting multiple input formats (BAM, VCF, FASTA, HSD, microarray) and providing both high-coverage (Kulczynski scoring) and low-coverage (tree traversal) classification methods.
- **Core Function**: Universal mtDNA haplogroup classifier for ancient and modern DNA
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda evehap`
