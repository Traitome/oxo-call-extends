## Concepts
- **Tool Overview**: A package for discovery, annotation, and analysis of gene clusters in genomics or metagenomics datasets.
- **Core Function**: Processes bioinformatics data for annotation tasks.
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ, BAM, VCF, etc.).
- **Installation**: `conda install -c bioconda opfi`
