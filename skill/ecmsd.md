## Concepts
- **Tool Overview**: ECMSD detects and taxonomically classifies mitochondrial sequences from high-throughput sequencing data using minimap2 alignment and RMUS scoring.
- **Core Function**: Efficient Comprehensive Mitochondrial Sequence Detection Pipeline
- **Input/Output**: Standard bioinformatics formats (FASTA/FASTQ/BAM/VCF/GFF)
- **Installation**: `conda install -c bioconda ecmsd`
